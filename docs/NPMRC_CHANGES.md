# Histórico de configurações do npmrc

- **Registro anterior:** não havia `.npmrc` no repositório nem em `~/.npmrc`, portanto o npm usava o registro padrão sem personalização.
- **Registro atual:** arquivo `.npmrc` no repositório define `registry=https://registry.npmjs.org/`.
- **Tokens:** nenhum token foi adicionado; caso seja necessário autenticar, use uma variável de ambiente (por exemplo, `//registry.npmjs.org/:_authToken=${NPM_TOKEN}`).
